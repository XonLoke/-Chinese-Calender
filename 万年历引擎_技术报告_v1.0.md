| `calendar/converters.py` | ~460 | Gregorian/Julian/ISO ↔ R.D. 双向转换 |

### 3.2 核心算法

**RataDie 历元约定：**
- R.D. 1 = 公元前 1 年 1 月 1 日（Gregorian proleptic）= Monday
- R.D. 1 对应 JD 1721424.5

**Moment：**
- R.D. 的浮点扩展，用于亚日精度
- `Moment.day` = R.D. + 日小数
- `Moment.to_datetime()` / `Moment.from_datetime()` ↔ Python datetime

**Gregorian ↔ R.D.（D&R Eq. 2.16-2.19）：**
```
fixed_from_gregorian(y,m,d):
  MonthDays = 前 (m-1) 个月天数 + d
  return 365×(y-1) + (y-1)//4 - (y-1)//100 + (y-1)//400 + MonthDays

gregorian_from_fixed(rd):
  prior_year = (rd - 1) // 146097 × 400  +  (日序推算)
```

**Julian ↔ R.D.（D&R Eq. 3.1-3.4）：**
```
fixed_from_julian(y,m,d):
  MonthDays = 前 (m-1) 个月天数 + d
  return 365×(y-1) + (y-1)//4 + MonthDays
```

**ISO ↔ R.D.（D&R Eq. 2.23-2.24）：**
- 每年第一周包含该年第一个周四
- 返回 (cycle, year, week, day) 四元组

### 4.2 时间系统

**JD ↔ Moment 转换：**
- `jd_from_moment(m)` = m.day + 1721424.5
- `moment_from_jd(jd)` = Moment(jd - 1721424.5)
- J2000.0: JD 2451545.0 = Moment 730120.5

**Julian Centuries：**
- `julian_centuries(m)` = (JD - 2451545.0) / 36525
- VSOP87 的标准输入参数 t

**ΔT（TT - UT）：**
```
ΔT(1620-1900): 8.83 + 0.0449 × (year - 1700)          [线性]
ΔT(1900-2000): -2.79 + 1.494119t - 0.0598939t² + ...  [四次]
ΔT(2000-2100): -1.15 + 0.0785t + 0.0004t²             [二次]
其中 t 在不同区间定义不同（详见代码）
ΔT(2000) ≈ 63.8 秒
```

**UTC ↔ TT 转换：**
- `dynamical_time(m)` → TT = UTC + ΔT/86400
- `universal_time(m)` → UTC = TT - ΔT/86400（2 次迭代）
- 往返误差 < 0.001 天

### 4.3 VSOP87 太阳黄经

**方案选择：** PyMeeus 封装（方案 A）

**核心函数：**
- `apparent_solar_longitude(m_tt)` → 太阳视黄经（度），已含岁差/章动/光行差
- `solar_longitude(m_tt)` → 几何黄经（未修正）
- `solar_longitude_rate(m_tt)` → 黄经变化率 ≈ 0.9856°/天（数值微分，1h 步长）

**Newton 法逆求解（节气判定核心）：**
```
solve_solar_longitude(target_lon, approx_moment):
  t_{n+1} = t_n - (λ(t_n) - λ_target) / λ'(t_n)
  收敛容差: 1e-8 天 (≈ 0.0009 秒)
  典型迭代: 3-5 次
```

| 黄经变化率 ≈ 0.9856°/天 | ✅ |

## 六、Sprint 4：新月合朔计算

### 6.1 交付物

| 文件 | 行数 | 核心内容 |

### 6.2 核心算法

**合朔条件：** `Moon.longitude(t) - Sun.longitude(t) ≡ 0° (mod 360°)`

**两阶段求解：**

```
Phase 1 — Meeus Ch.47 平均新月（初始猜测，~15 分钟精度）
  k = (JD - 2451550.09765) / 29.530588861
  JDE_mean = 2451550.098 + 29.530588861·k + 0.0001337·T² + ...
  JDE≈ = JDE_mean + Σ(周期修正)     ← ~20 项摄动项

Phase 2 — Newton 法精化（< 0.1 秒精度）
  f(t) = Moon_lon(t) - Sun_lon(t)  → 0
  f'(t) ≈ 12.19°/day               ← 数值微分
  t_{n+1} = t_n - f(t_n) / f'(t_n)
  收敛条件: < 1e-6° (≈ 0.1 秒时间)
  典型迭代: 2-3 次
```

**关键参数：**
- 参考新月 JDE: 2451550.09765（2000年1月6日）
- 朔望月: 29.530588861 天
- 日月相对运动速率: ~12.19°/天

### 6.3 公开 API

- `new_moon_after(moment)` — 给定时刻后的第一个新月（UTC）
- `new_moon_before(moment)` — 给定时刻前的最后一个新月（UTC）
- `new_moon_at_lunation(bln)` — 指定 Brown Lunation Number 的新月
